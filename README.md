## 💡 프로젝트 소개
- 세종대학교 공학인증 상태 정보를 제공해주는 서비스입니다.
- 학생별 기이수 강의목록 엑셀파일을 업로드하여 학생의 학과와 학년에 맞게 공학인증 상태를 보여주며, 추가로 들을 수 있는 강의 목록을 제공합니다.
